## Project Overview

End-to-end supervised learning pipeline that predicts wine quality (binary: high ≥ 6 / low < 6) from 11 physicochemical measurements using the [UCI Wine Quality Dataset](https://archive.ics.uci.edu/dataset/186/wine+quality). Three classifiers are compared under a strict no-leakage protocol with class-imbalance-aware tuning, statistical significance testing, multi-seed robustness, and an all-model feature-engineering ablation.

**Headline result:** Random Forest — Test ROC-AUC **0.904 [95% CI: 0.883, 0.922]**, Test PR-AUC **0.938**, Brier **0.124** (best calibrated). All pairwise differences statistically significant (McNemar p < 0.001).

**Notable secondary result:** SVM with balanced class weights achieves the highest low-quality recall (0.796), making it the recommended model when minority-class detection is the priority.

## How to Run

### 1. Run the full pipeline (recommended)

```bash
python3 final_pipeline.py
```

This single command will:
1. Download `winequality-red.csv` and `winequality-white.csv` from UCI (cached in `data/`)
2. Run preprocessing, feature engineering, and VIF analysis
3. Create stratified 70/15/15 splits (seed 42)
4. Tune all three models via 5-fold stratified CV — including `class_weight ∈ {None, "balanced"}` for all three
5. Compute test metrics with **95% bootstrap CIs**, **PR-AUC**, and **Brier scores**
6. Run **McNemar's test** for pairwise model comparisons
7. Run **multi-seed robustness study** (5 stratified splits)
8. Run **all-model ablation**: base (11) vs engineered (14) features for each model
9. Run **threshold tuning** for Logistic Regression
10. Run **density-drop experiment** to confirm VIF finding
11. Save 10 figures to `figures/` and dump all numbers to `results.json`

**Expected runtime:** 6–12 minutes on a modern laptop.

### 2. Rebuild the PPTX deck

```bash
node build_slides.js
```

### 3. Reproducibility

All paths in `final_pipeline.py` are now relative to the script's directory (`os.path.dirname(__file__)`), so the script runs unchanged on any machine after `pip install -r requirements.txt`.

## Experimental Design Notes

- **No data leakage:** `StandardScaler` and outlier percentiles fit on training set only; test set untouched until final eval.
- **Hyperparameter tuning:** 5-fold stratified CV inside training fold; class_weight tuned for **all three models**.
- **Test set protocol:** Each model's `predict` and `predict_proba` is called on the test set exactly once.
- **Stratified splits:** All partitions maintain the 63.3% / 36.7% class ratio.
- **Statistical rigor:** Bootstrap CIs, McNemar's test, and multi-seed robustness used jointly to confirm the model ranking is real, not seed-dependent.
